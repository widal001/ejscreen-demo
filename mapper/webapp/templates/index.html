EJScreen Demo

This open source project replicates most of the functionality of the EPA's EJSCREEN mapping tool with an updated look and feel. It also provides machine readable access to the underlying data through a REST API.

Start mapping »

Original Tool

The core feature set and underlying data for this project were sourced from the Environmental Justice Screening and Mapping Tool. To learm more about the original tool and how it was made, visit the EPA's website.
